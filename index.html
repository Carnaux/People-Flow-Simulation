ROOM
Github
Dynamically simulate people flow in buildings
Upload any type of building.
To navigate the people spawned the software uses navmeshes.
Create entrances and exits where you want
The simulation is based on points of interest. The user will set them accordingly to what is being simulated, for example, a shopping will have clothing stores, movies, food etc. Each person spawned has a number linked to a interest, the highest number determines where he will go.
It is possible to add points of interest the size of shelf items
It is possible to add points of interest the size of rooms
After a person satifies one of the interests he will go to the next one or go home, this decision is randomly generated.
All images are artistic renderings and they were inspired by the music of Kyle Dixon & Michael Stain
